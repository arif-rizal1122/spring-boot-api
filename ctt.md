# Spring RESTful API

¥ Requirement
1. Pada kelas ini, kita akan coba membuat RESTful API untuk Contact Management, dimana RESTful API yang akan kita buat memiliki fitur sebagai berikut :
 - User Management
 - Contact Management
 - Address Management

# User Management Requirement

¥ User Data
- Username
- Password
- Name

¥ User Api
- Register User
- Login User
- Update User
- Get User
- Logout User

# Contact Management Requirement

¥ Contact Data
- First Name
- Last Name
- Email
- Phone


¥ Contact API
- Create Contact
- Update Contact
- Get Contact
- Search Contact
- Remove Contact

# Address Management Requirement

¥ Contact Address Data
- Street
- City
- Province
- Country
- Postal Code

¥ Address API
Create Address
Update Address
Get Address
List Address
Remove Address
